<objective>
Add the genuinely-new statistic this milestone needs: a **paired-bootstrap confidence
interval** on per-trial judge deltas (arm vs the `none` baseline), as a small seeded
stdlib helper, and wire the CI'd delta into the Track-2 judge output produced by
`replicate.py`.

Purpose: Wilson (`grounding.py::_wilson`) and Cohen's d (`replicate.py::_cohens_d`)
already exist and are neither a paired-bootstrap CI. HAR-04 says reuse the multi-trial
loop and the CI idiom — do not rebuild them — and add the paired-bootstrap as a helper
that consumes the SAME per-trial judge-score trajectories `replicate.py` already
aggregates. This plan delivers the helper + the Track-2 wiring (criterion 1). Plan 02's
one-command driver and Plan 03's E2E smoke both build on this.

Scope note (revision): the milestone's user-facing surface for the CI'd delta is
`replicate.py`'s own summary JSON (`bootstrap_ci_delta_vs_none`) plus Plan 02's
`close_loop` JSON. Per CLAUDE.md "no features beyond what was asked", this plan does NOT
touch `bench/report.py` — adding a `judge_ci` param there would be dead wiring, since no
caller in this phase routes the CI through `report.write_json`.

Output: `bench/bootstrap.py` (new) and CI'd deltas threaded into `replicate.py`'s summary
JSON, with determinism + edge-case tests.
</objective>

  <behavior>
    - Deterministic: `paired_bootstrap_ci([...], seed=_BOOTSTRAP_SEED)` returns identical mean/ci_low/ci_high on repeated calls (asserted byte-identical across two calls).
    - Paired input: consumes a single list of per-trial paired deltas (arm_improvement_t - none_improvement_t) and returns the 95% percentile CI of the resampled mean.
    - Edge cases: empty list -> {"n": 0, "mean": None, "ci_low": None, "ci_high": None}; n==1 -> mean present, ci_low==ci_high==mean (degenerate, never raises); all-equal deltas -> zero-width CI at that value.
    - ci_low <= mean <= ci_high always; changing the seed changes the bounds but not the mean.
    - Never raises on any numeric input (mirrors grounding/replicate never-raises discipline).
  </behavior>
  <action>
    Create `bench/bootstrap.py` (module docstring stating: Track-2 only, stdlib-only, seeded,
    never-raises; the paired-bootstrap CI is EXCLUDED from the mechanical compounding_score).
    Define `_BOOTSTRAP_SEED = 1729` (named module constant) and `_DEFAULT_RESAMPLES = 2000`.
    Implement `paired_bootstrap_ci(deltas, *, resamples=_DEFAULT_RESAMPLES, seed=_BOOTSTRAP_SEED,
    confidence=0.95) -> dict` returning keys `{"n", "mean", "ci_low", "ci_high", "resamples",
    "seed", "confidence"}`. Use a local `random.Random(seed)` instance (NEVER module-level
    `random.random()` — determinism requirement from critical domain fact 1) to draw `resamples`
    with-replacement samples of size n from `deltas`, take each resample's `statistics.mean`,
    sort, and read the percentile bounds at `(1-confidence)/2` and `1-(1-confidence)/2`. Round
    bounds/mean to 2 decimals to match replicate.py's rounding convention. Import only `random`
    and `statistics` from stdlib.
  </action>
