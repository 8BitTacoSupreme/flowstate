<objective>
Build a NEW module `bench/grounding.py`: a checkable grounding-eval harness that
replaces vibe-scoring (the 0-10 judge) for measuring context-layer value. Per
(arm, probe): inject the arm's context prefix, ask the probe question, then K
judges binary fact-check the answer against ground truth. Arm score = grounding
accuracy (% probes majority-correct), with Wilson score CIs — far lower variance
than the 0-10 judge.

Purpose: give a high-N, binary, multi-judge signal for whether an injected context
arm lets the model state SPECIFIC facts correctly.

Output: `bench/grounding.py`, an example probes fixture, and a fully-mocked test
suite (no live claude calls).

ADD-ONLY: do NOT modify pipeline, judge, replicate, compound_eval, or context_prefix.
Research tooling only — no UI, never-raises throughout, stdlib only
(math/json/subprocess/argparse/re/os/sys).
</objective>

    IMPORTS (reuse, never redefine): `from bench.compound_eval import _LAYERS_MAP`,

    CONSTANTS: `_ANSWER_MAX_ATTEMPTS = 3`, `_ANSWER_TIMEOUT = 180`, `_JUDGE_TIMEOUT = 60`.
    Compile a module-level regex for the first YES/NO token, e.g.
    `_YESNO_RE = re.compile(r"\b(yes|no)\b", re.IGNORECASE)`.

    PROBES LOADER `_load_probes(path: Path) -> list[dict] | None` (never-raises): read the
    JSON file; expect a list of dicts with keys "id","question","ground_truth" (optional
    "source"). On missing file, parse error, non-list, or empty list -> return None (caller
    prints message + returns non-zero). Must make NO subprocess calls. Catch all exceptions.

    HELPER `_wilson(successes: int, n: int) -> tuple[float, float]`: Wilson score interval
    at z=1.96 using stdlib math only. n==0 -> (0.0, 0.0). Compute phat=successes/n; center
    and half-width per the standard Wilson formula; clamp both bounds to [0,1]; return
    (low, high). Round is optional but bounds MUST satisfy low<=phat<=high.

    HELPER `_answer(prefix: str, question: str, model: str) -> str` (never-raises): if
    `_locate_claude()` is None -> return "". Build prompt =
    `(prefix + "\n\n---\n\n" if prefix else "") + "Question: " + question + "\nAnswer concisely and specifically."`.
    Command: `[claude, "--print", "--max-turns", "1", "--model", model, "--", prompt]`.
    Loop up to `_ANSWER_MAX_ATTEMPTS`: `subprocess.run(cmd, capture_output=True, text=True, timeout=_ANSWER_TIMEOUT)`;
    on returncode == 0 AND stdout.strip() -> return stdout.strip(). On returncode!=0,
    empty stdout, or exception -> continue. Return "" if all attempts fail. Mirror
    research.py's empty-then-good retry exactly.

    HELPER `_factcheck(answer: str, ground_truth: str, model: str) -> bool | None`
    (never-raises): build prompt =
    `"Does the ANSWER correctly state this FACT? Reply with ONLY 'YES' or 'NO'.\n\nFACT: " + ground_truth + "\n\nANSWER: " + answer`.
    If `_locate_claude()` is None -> return None. Command:
    `[claude, "--print", "--max-turns", "1", "--model", model, "--", prompt]`,
    `subprocess.run(..., timeout=_JUDGE_TIMEOUT)`. Parse first YES/NO token via `_YESNO_RE`
    against stdout (lowercased): "yes" -> True, "no" -> False. Unparseable / empty / non-zero
    / exception -> None. (No retry needed; one call.)

    PER-PROBE EVALUATION inside main's nested loop (trial x arm x probe):
      1. `with MemoryStore(root=root) as mem: prefix = build_context_prefix(root, mem, query=probe["question"], include_layers=_LAYERS_MAP[arm])`.
         (`_LAYERS_MAP["full"]` is None = all layers; "none" = frozenset() = empty prefix.)
      2. `answer = _answer(prefix, probe["question"], answer_model)`.
      3. If `answer == ""`: SKIP all `_factcheck` calls; votes = [None]*len(judge_models),
         treat majority as False (all-NO). Else `votes = [_factcheck(answer, probe["ground_truth"], m) for m in judge_models]`.
      4. `yes = sum(1 for v in votes if v is True)`; `majority = yes > len(judge_models) / 2`
         (None counts as NO).
      5. Record `{"arm": arm, "trial": trial, "probe_id": probe["id"],
         "answer_chars": len(answer), "votes": ["YES" if v is True else "NO" if v is False else "?" for v in votes],
         "majority": majority}`.

    AGGREGATION per arm over all (probe x trial) records: `n = len(records)`;
    `successes = sum(1 for r in records if r["majority"])`; `accuracy = successes/n if n else 0.0`;
    `wilson_low, wilson_high = _wilson(successes, n)`. Build
    `arms[arm] = {"accuracy": accuracy, "n": n, "wilson_low": ..., "wilson_high": ..., "per_probe": records}`.
    Compute `accuracy_delta_vs_none = {a: round(arms[a]["accuracy"] - arms["none"]["accuracy"], 3) for a in arms}`
    ONLY when "none" in arms (else omit / empty dict).

    CLI `_build_parser()` (argparse, prog="bench.grounding"):
      `--root` (type=Path, required=True); `--probes` (type=Path, required=True);
      `--layers` (nargs="+", choices=("full","none","pack","memory","wiki"), default=["none","pack","wiki"]);
      `--trials` (type=int, default=2); `--answer-model` (default="sonnet");
      `--judge-models` (str, default="sonnet,sonnet,opus"); `--budget-tokens` (type=int, default=50000);
      `--out` (type=Path, default=None).

    `main(argv=None) -> int`: parse args; FIRST set
    `os.environ["FLOWSTATE_CONTEXT_BUDGET_TOKENS"] = str(args.budget_tokens)` (so
    build_context_prefix honors it). Load probes via `_load_probes`; if None -> print a
    clear message (e.g. "no usable probes in <path>") and return 1 BEFORE any subprocess call.
    Parse judge_models = `[m.strip() for m in args.judge_models.split(",") if m.strip()]`.
    Run nested loop trials x args.layers x probes, collecting per-arm records. Aggregate.
    Build output dict: `{"probes_file": str(args.probes), "n_probes": len(probes),
    "trials": args.trials, "answer_model": args.answer_model, "judge_models": judge_models,
    "arms": arms, "accuracy_delta_vs_none": accuracy_delta_vs_none}`.
    If `args.out`: write JSON (never-raises — on OSError/exception print a warning, do not crash).
    Always print a compact console summary table (arm | accuracy | wilson CI | n) — plain
    print() / formatted strings are fine (no Rich dependency required; matches stdlib-only
    constraint). Return 0.

    Footer: `if __name__ == "__main__": sys.exit(main())`.

    `{"id": "q1", "question": "...", "ground_truth": "...", "source": "..."}`. Use neutral
    example facts (e.g. project name, default port, a config key) — purely illustrative.
