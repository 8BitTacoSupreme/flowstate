## What Was Built

### bench/grounding.py additions

**Constants:**
- `_REFUSAL_PHRASES` frozenset — fast-path substring set for `_judge_rejection`
- `_RGB_SEP`, `_RGB_NEGATIVE_INSTRUCTION`, `_RGB_COUNTERFACTUAL_INSTRUCTION` — axis-specific prompt variants

**Modified:**
- `_answer(prefix, question, model, *, instruction="Answer concisely and specifically.")` — keyword-only `instruction=` kwarg; default reproduces original trailer byte-for-byte; all existing positional callers unaffected

**New helpers (never-raise, stdlib-only, deterministic):**
- `_judge_rejection(answer, model)` — regex fast-path on `_REFUSAL_PHRASES` → True with zero subprocess; fall-through to one-shot LLM YES/NO judge
- `_rgb_distractors(probe, probes, n)` — gold passages from other probes, sorted by id, self excluded, list gold expanded
- `_rgb_noise(probe, probes, noise_ratio, k, answer_model, judge_models)` — gold always present + floor(ratio×k) distractors
- `_rgb_negative(probe, probes, k, answer_model, judge_models)` — distractors only; scored by rejection rate
- `_rgb_integration(probe, probes, k, answer_model, judge_models)` — multi-gold probes only; returns None (skipped) otherwise
- `_rgb_counterfactual(probe, answer_model, judge_models)` — counterfactual+wrong_answer probes only; returns None otherwise

**New dispatcher:**
- `_run_rgb(args, probes)` — sweeps noise_ratios, iterates all axes, aggregates with Wilson CIs, writes JSON, prints console table; returns 1 only if all axes have zero records

**CLI additions (additive):**
- `--mode {layers,rgb}` default `layers`
- `--axes` default `noise,negative,integration,counterfactual`
- `--noise-ratios` default `0.0,0.4,0.8`
- `--rgb-k` type int, default 5

**main() dispatch:** `if args.mode == "rgb": return _run_rgb(args, probes)` placed immediately after the probes guard; the arm loop block is reached only when mode != rgb and is byte-for-byte unchanged.
