<objective>
Add deterministic, caller-driven supersession to the SQLite memory store so retrieval injects current truth instead of stale memories. Additive schema column (`superseded_by`), an explicit `supersede()` API, superseded-row exclusion from retrieval (byte-identical when unused), and a flag-only `find_contradiction_candidates()` that reuses existing vectors.

Purpose: Let callers mark a memory as replaced so `get_context()` stops injecting stale facts — without any LLM contradiction detection (preserves the "memory is pure-Python, no LLM" Key Decision) and without ever auto-deleting/auto-superseding (honors the ECC silent-content-loss caution).

Output: Modified `flowstate/memory.py` + new `tests/test_memory_supersession.py`.
</objective>

SCHEMA_SQL: `CREATE TABLE IF NOT EXISTS memories (...)` with columns
  rowid, id, kind, content, summary, source, tags, metadata, created_at, run_id.
  `schema_version` table holds a single INTEGER version row (currently 1).
  `memories_fts` is external-content FTS5 over (summary, content, tags) with
  ai/ad/au triggers — it references ONLY those three columns, so adding a
  column to `memories` requires NO FTS schema/trigger change.

__init__ order (lines ~168-183):
  connect → row_factory=Row → executescript(SCHEMA_SQL) → set _embedder →
  _init_vec() → _backfill_pending. The migration hook goes AFTER executescript,
  BEFORE _init_vec().

MemoryEntry (dataclass): id, kind, content, summary, source, tags, metadata,
  created_at, run_id. `create()` classmethod builds active entries.

_row_to_entry(row) -> MemoryEntry — maps a sqlite3.Row to MemoryEntry.

search(query, *, kind=None, limit=10) -> list[SearchResult]
  — two FTS join branches (with/without kind); score = abs(row["rank"]).

_semantic_results(query, k) -> list[SearchResult] | None
  — vec0 KNN: `SELECT rowid, distance FROM memories_vec WHERE embedding MATCH ?
    ORDER BY distance LIMIT ?`, filters by _SEMANTIC_MAX_DISTANCE, then resolves
    each rowid via `SELECT * FROM memories WHERE rowid=?`. score = L2 distance.

get_context(query, *, max_tokens=2000, k=_SEMANTIC_K) -> str
  — _semantic_results() then falls back to search(); formats "## Prior Knowledge".

_embedder: Embedder with `.available()` and `.embed([texts]) -> list[list[float]]`.
  _vec_ready: bool. sqlite_vec.serialize_float32(vec) for blob binding.
  _SEMANTIC_MAX_DISTANCE = 0.89 (L2; bge unit-normalized, L2²=2(1−cos)).

Test idioms (tests/test_memory.py): `_fake_embedder(dim=4)` deterministic embed_fn;
