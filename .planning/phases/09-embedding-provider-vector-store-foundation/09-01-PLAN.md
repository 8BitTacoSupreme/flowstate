<objective>
Create the optional embedding provider module `flowstate/embeddings.py` and declare fastembed as
an opt-in `[semantic]` pip extra. This is the embedding half of the Phase 9 foundation — the vector
store (Plan 09-02) consumes this provider.

The provider ports `_default_embedder` from `bench/grounding.py` into a reusable shape: a lazy
fastembed-backed embedder that (a) imports cleanly without fastembed installed, (b) reports
availability without raising, (c) caches the loaded model, and (d) resolves the model name via the
same env > config.json > default precedence used by `context_prefix._load_budget`.

Purpose: graceful-degradation is the headline guarantee. Core install stays dependency-free; the
embedder activates only when the user opts into `[semantic]`. When absent, every downstream caller
sees `available() == False` and does nothing.

Output: `flowstate/embeddings.py`, the `[semantic]` extra in `pyproject.toml`, and
`tests/test_embeddings.py` (no model/network required).
</objective>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| fastembed first-use → HuggingFace | First `embed()` with the real provider downloads ~130MB `BAAI/bge-small-en-v1.5` from a remote host (supply-chain / offline risk) |
| optional-extra opt-in | The core install never imports or downloads fastembed; only `pip install flowstate[semantic]` + an explicit embed call reaches the network |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-01 | Tampering | fastembed model download (HuggingFace) | accept | Blast radius contained by design: download happens only on opt-in `[semantic]` install AND first embed; default path is offline. fastembed pins its own model hashes. No mitigation code in this phase. |
| T-09-02 | Denial of Service | fastembed import/load failure | mitigate | Lazy import wrapped in try/except → `available()` returns False, `embed()` returns []; never blocks or raises. Covered by the import-without-fastembed test. |
| T-09-SC | Tampering | pip install (fastembed in [semantic] extra) | mitigate | fastembed is the only new install and is opt-in. See Package Legitimacy note below; pin `>=0.3`. No `[ASSUMED]`/`[SUS]` blocking checkpoint required because the extra is not installed by the core/default path and tests never install it. |
