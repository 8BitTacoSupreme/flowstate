| corrupt memory.db → CLI | a malformed DB file must degrade gracefully, never traceback |
| doctor findings → capture | diagnosis messages are local but feed capture/SQLite |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-07-09 | Denial of Service | corrupt/missing memory.db crashing the CLI | mitigate | try/except around MemoryStore open in both list and prune → friendly message, exit 0 |
| T-07-10 | Denial of Service | doctor/repair capture raising and breaking the command | mitigate | best-effort try/except: pass around the capture block; capture_gotcha itself never-raises |
| T-07-11 | Information Disclosure | gotcha messages echoed in table | accept | Local-only data already in memory.db; no new exposure surface |
