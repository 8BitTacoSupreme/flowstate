<objective>
Close HAR-02: an arm whose REQUIRED producer artifact is absent must fail loud —
never report a bare number for an arm that had no input. Today `bench/compound_eval.py`
happily runs `--layers wiki` with no wiki corpus (the reader returns "" and the arm
"measures nothing"), and `--layers pack` with no repomix pack. Phase 16 already
threaded `producers` / `layers_present` provenance into the report; this plan builds a
HARD gate on top of that provenance: before (or at) the point a number is attributed to
an arm, verify the arm's producer artifact exists in `--root`. If missing, emit a
prominent "arm measured nothing: producer X absent" marker AND exit non-zero.

Purpose: no silent no-op arms — a benchmark you cannot trust is worse than none.
Output: fail-loud gate in `compound_eval.py` + regression tests.
</objective>
