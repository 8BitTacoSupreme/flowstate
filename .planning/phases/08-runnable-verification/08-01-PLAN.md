    Create `flowstate/verify.py` with `from __future__ import annotations` header and a module docstring stating pure-Python, no LLM. Imports: `json`, `logging`, `re`, `xml.etree.ElementTree as ET`, `from dataclasses import dataclass`, `from pathlib import Path`, `from typing import Literal`, and `from flowstate.state import FlowStateModel`. Do NOT import `flowstate.bridge` (LOCKED constraint).

    Define `@dataclass(frozen=True) class VerifyResult` with fields `gate: str`, `status: Literal["pass", "fail", "skip"]`, `message: str`, `fixture: str` — mirror the Diagnosis shape from doctor.py.

    Define module constant `_COVERAGE_RE = re.compile(r"coverage meets or exceeds (\d{1,3})%", re.IGNORECASE)` — bounded `{1,3}` digit quantifier is ReDoS-safe and matches the context.py L189 gate text "Test coverage meets or exceeds {N}%".

    Define `_parse_coverage_rate(root: Path) -> float | None`: read `root / "coverage.xml"`; if absent return None; else `try: tree = ET.parse(str(cov_xml)); rate = tree.getroot().get("line-rate"); return float(rate) if rate is not None else None` wrapped in `except Exception: return None`. Never shell out to `coverage report`; `.coverage` is a binary/SQLite file and is NOT parsed (no `coverage` package — banned new dep). Returns a 0.0–1.0 line-rate or None.

    Define `_check_artifact_integrity(state, root) -> list[VerifyResult]`: loop `state.install_manifest`; skip entries where `entry.checksum is None`; for each remaining entry compute `path = root / entry.path`; if `not path.exists()` append `VerifyResult(gate="produced-artifact-integrity", status="fail", message=f"Produced artifact missing: {entry.path}", fixture="(manifest)")`; elif `path.is_file() and path.stat().st_size == 0` append a fail with message "Produced artifact empty: {entry.path}". This backbone check runs once per verify regardless of gate text.

    Define `_check_coverage_gate(gate, root, fixture_name) -> VerifyResult`: match `_COVERAGE_RE` against the gate string; the caller only invokes this when matched. Parse the required N from the match group (int). Call `_parse_coverage_rate(root)`; if None → `VerifyResult(gate, "skip", "no coverage report found (coverage.xml absent)", fixture_name)`; else compute `actual_pct = rate * 100`; if `actual_pct >= N` → status "pass" with message like "coverage {actual_pct:.1f}% >= {N}%"; else status "fail" with message "coverage {actual_pct:.1f}% < {N}%".

    Define `run_verify(state: FlowStateModel, root: Path) -> list[VerifyResult]` mirroring run_doctor's structure: start `results: list[VerifyResult] = []`; run `_check_artifact_integrity(state, root)` once inside a try/except (on exception append a skip result, never raise). Then glob `sorted((root / ".planning" / "fixtures").glob("*.json"))`; for each fixture path, in a per-fixture `try/except Exception` block: read text, `json.loads`, extract `acceptance_gates` (list, default []) and `forbidden_actions` (list, default []); for each acceptance_gate string: if `_COVERAGE_RE.search(gate)` matches → append `_check_coverage_gate(gate, root, fixture_path.name)`, else append `VerifyResult(gate, "skip", "not mechanically verifiable (manual gate)", fixture_path.name)`; for each forbidden_action string append `VerifyResult(action, "skip", "not mechanically verifiable (forbidden action)", fixture_path.name)`. On a per-fixture exception, `logger.warning(...)` and append `VerifyResult(gate=fixture_path.name, status="skip", message=f"malformed fixture skipped: {e}", fixture=fixture_path.name)` then continue. The whole function is self-contained never-raises. Return `results`.

    Add `logger = logging.getLogger(__name__)` at module level. Keep line-length ≤ 100, double quotes, snake_case.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| fixture JSON files → run_verify | Untrusted/adversarial on-disk content crosses into the parser |
| coverage.xml → _parse_coverage_rate | Possibly malformed/adversarial XML crosses into ET.parse |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01 | Denial of Service | _COVERAGE_RE regex on fixture gate text | mitigate | Bounded `(\d{1,3})` quantifier, no nested/backtracking groups — ReDoS-safe; `.search` on plain strings only |
| T-08-02 | Tampering | malformed/huge/adversarial fixture JSON | mitigate | Per-fixture try/except: json.loads failure → skip-result + logger.warning, never raise; run_verify self-contained never-raises |
| T-08-03 | Tampering | malformed coverage.xml (XML-bomb / bad attrs) | mitigate | _parse_coverage_rate wraps ET.parse + float() in try/except → returns None → SKIP; defusedxml unavailable so guard via except |
| T-08-04 | Information Disclosure | fixture/coverage reads escaping project root | accept | All reads built from `root / ...` literals (no path traversal from external input); manifest paths are FlowState-authored, not user input |
| T-08-05 | Tampering | empty-file artifact passed off as valid produced output | mitigate | Integrity check FAILs on `st_size == 0`, not just existence |
</threat_model>
