<objective>
Ship the ONE command HAR-04 sentence 2 / criterion 2 demands: a single entry point that
chains **prior-runs → distill → inject → judge → paired-bootstrap CI** on a fixture and
returns a CI'd delta (not a single-shot score), reusing the Phase-17 producers and the
Plan 01 helper rather than rebuilding anything.

Purpose: Close the v0.6.2 "Make the Harness Real" loop — the harness can now run the full
compounding-value chain end to end and report a confidence-interval'd result. Per the
critical domain facts, this picks the SMALLER surface (a dedicated thin driver module that
composes existing entry points) and provides a `--mode cheap` path so the loop is provably
CI-safe with no live LLM.

Revision note — the fixture-seeding fix: `bench/fixtures/sample_project` has NO `memory.db`,
and the Phase-17 distiller correctly fails loud (rc=1) on empty memory. So `close_loop` MUST
NOT run the distiller directly against `--root`. Instead it mirrors `compound_eval._worktree`:
copy `--root` into an isolated temp worktree, `bench.project.scaffold(target)` to seed a
baseline `MemoryKind.RUN` entry (non-empty memory), and run DISTILL/INJECT-PREP against that
copy. The checked-in fixture is never touched, and the wiki producer always has distillable
memory — so the `--mode cheap` verify command exits 0.

Output: `bench/close_loop.py` with a `main(argv)` entry point, invocable as
`python -m bench.close_loop --root <fixture> --mode cheap`, plus a CI-safe test.
</objective>

bench/compound_eval.py (the isolation pattern to MIRROR):
  - `_worktree(root) -> Iterator[Path]` — contextmanager; `shutil.copytree(root, target)` into a
    fresh `tempfile.mkdtemp` dir, yields the copy, `shutil.rmtree(..., ignore_errors=True)` on exit.
    Source `root` is NEVER written to. `_cheap_loop` (lines 246-268) shows the canonical usage:
    `with _worktree(root) as target: scaffold(target); ...`.

bench/project.py (fixture seeding — gives the distiller non-empty memory):
  - `scaffold(root, *, synthetic=True) -> None` — seeds a self-consistent synthetic baseline and,
    via `_seed_baseline_run` (lines 162-189), writes a baseline `MemoryKind.RUN` entry into
    `root/memory.db`. After `scaffold(target)`, the target has distillable memory.

bench/prepare_fixture.py::main(argv) -> int
  - `main(["--root", str(target), "--arms", "wiki"])` distills memory→wiki corpus via
    bench.distiller and (for pack) flowstate.pack.run_pack. Returns 0 iff all producers succeed.
    Requires non-empty memory for the wiki producer — hence the scaffold-seeded worktree above.

bench/distiller.py::main(argv) -> int
  - `--root <path>` [--force] [--llm --model <m>]; writes `.planning/codebase/wiki/` corpus.
    Fails loud (rc=1) on empty memory — intentional Phase-17 behavior; do NOT call against a
    bare fixture.

bench/replicate.py (multi-sample judge trajectories to REUSE):
  - `_run_trial(arm, runs, root, label) -> list[float] | None` — one compound_eval real-mode
    invocation, returns per-run judge scores (live LLM). Monkeypatch this in tests for CI-safety.
  - `_agg(trials) -> dict` returns `improvements` (per-trial last-first list).

bench/bootstrap.py (Plan 01):
  - `paired_bootstrap_ci(deltas, *, resamples=2000, seed=_BOOTSTRAP_SEED, confidence=0.95) -> dict`
    with keys n/mean/ci_low/ci_high/resamples/seed/confidence.

bench/compound_eval.py:
  - `--layers` arms: full|none|pack|memory|wiki; `--mode {cheap,real}`; judge requires
    `--mode real --allow-llm`; `_EXIT_PRODUCER_ABSENT = 3`.
